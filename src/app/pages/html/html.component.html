HTML

HyperText Markup Language es un lenguaje de marcado que define la estructura de tu contenido. Utiliza etiquetas (tags) para enmarcar y definir cada parte del contenido. También se pueden incluir atributos en las etiquetas para definir mejor o personalizarlas.

HTML master tag reference w3schools

Estructura

<!DOCTYPE html>
Marca HTML, se pone al principio de todo para indicar que el codigo es HTML5.
<html>...</html>
Código, indica el principio y final del código HTML.
lang="..." (es, en-US, cat...) Atributo que indica el lenguaje del contenido.
<head>...</head>
Metadatos, se incluen etiquetas con metadatos que ayudan y dan mas información acerca del contenido que vamos a encontrar.
<meta>
Metadata, incluimos los datos de metada con atributos.
charset="..." (UTF-8) Codificación de los caracteres, que simbolos o caracteres se van a utilizar, ayuda a escritura con tildes.
name="..." content="..." (author + Marcos, Keywords + Crack, viewport + width=device-width, initial-scale=1.0 (ancho + ancho del dispositivo y escala 1 ) Se indica el name el tipo de información que se va a dar y en content de defina la información
<title>...</title>
Título, texto que aparecera como título de nuestra página en la pestaña.
<link...>
Sirve para lincar a otros archivos o documentos. Útil para archivo CSS con atributos y url con fuentes tipograficas en linea:
rel="stylesheet" "href="style.css"
<body>...</body>
Contenido, cuerpo de la página donde incluiremos todo el contenido a mostrar.
<header>...</header>
Cabecera, suele contener marca y/o logo que linka a la pagina inicial.
<nav>...</nav>
Navegación, suele contener el menu de navegación de la página.
<article>...</article>
Articulo, suele contener contenido independiente, puede haber multiples articulos si es necesario (Ej. noticias).
<section>...</section>
Sección, diferencia grandes partes del contenido y lo divide en secciones (Ej. capitulos, categorias).
<main>...</main>
Contenido principal, marca el contenido principal de la página.
<div>...</div>
División, sirve para diferenciar diferentes partes del contenido.
<p>...</p>
Paragrafo, engloba el conjunto de el texto y realiza un salto de linia al finalizar.
<span>...</span>
División 2, útil para clasificar o diferenciar texto con ciertos atributos.
<aside>...</aside>
Extra info, puede contener algo no relacionado con el contenido como, banner, publicidad, etc.
<hr>
Linea horizontal, dibuja una linea en todo lo ancho y salto de linea.
<br>
Break, salto de linea
<footer>...</footer>
Pie de pàgina, suele contener marca con © 2021, ayuda, privacidad, contacto, etc.

Atributos generales

id="..." (NOMBRE) identificador único de la etiqueta, muy útil para referirse a este contenido en CSS o JS.
class="..." (NOMBRE) identificador que puede repetirse en multiples etiquetas, muy útil para referirse a este contenido en CSS o JS. Se pueden incluir varios identificadores separandolos con un espacio dentro del mismo atributo " ". Ej. class="icon nav menu"
style="..." (CSS) estilos a aplicar solo en esta etiqueta. Mejor siempre crear archivo .css con los estilos a parte.

Listas

<ul>...</ul>
Lista NO ordenada, se clasifica los elementos con el mismo distinguido.
<li>...</li> Elemento de la lista, enmarca cada elemento de la lista.

<ol>...</ol>
Lista ordenada, se clasifica los elementos ordenandolos de primero a último.
type="..." (1, A, I) indica que numero o letras queremos para ordenar.
start="..." indica el numero por el que empieza el primer elemento.
<li>...</li> Elemento de la lista, enmarca cada elemento de la lista.

<dl>...</dl>
Lista de definiciones, permite crear una lista con elementos a definir, diferenciando elemento a definir de definición.
<dt>...</dt> Elemento a definir.
<dd>...</dd> Definición del elemento.

Multimedia

Formatos de archivos soportados:
  • Fotografia: JPEG, GIF, PNG, APNG, SVG (vectorial), BMP
  • Audio: MP3, WAV, OGG.
  • Video: MP4, WEBM, OGG
<img...> Inserta la fotografia indicada en los atributos.
src="..." Nombre del archivo y ruta dentro de la carpeta o URL.
alt="..." Título de la imagen.
width="..." Ancho en px
height="..." Alto en px
loading="..." Modo de carga de las imágenes. Por defecto se cargan todas, si marcamos este atributo como lazy se cargará cuando la imágen tenga que aparecer en pantalla.

*Importante saber elegir formato y tamaño de la imagen, los mas importantes: JPEG permite comprimir bastante, útil para fotografias, gradientes. PNG tiene menos perdida de info y permite canal alpha, útil para transparencias, textos, monocolor. GIF solo permite 256 colores pero puede guardar varios frames creando imagenes en movimiento, útil para animaciones cortas.

<picture>...</picture> Define el espacio para multiples imagenes. Se usa el que encaja en el pagina cargada. Las imagenes contenidas se definen con <img> o <source>

<audio>...</audio>
<source src="..." type="..." Se indica el nombre del archivo o ruta + el formato (video/mp4).
controls añade controles de reproducción.
autoplay Se reproduce al cargar la página.
loop Se reproduce en bucle.
muted Se reproduce sin sonido.

*Se pueden añadir varios <source src="..." type="..." > con diferentes formatos por si el navegador no reconoce algún formato, eligirá siempre el primer formato reproducible. También se puede incluir un mensaje de error antes de cerrar etiqueta </audio> para que se muestre si no se puede reproducir ningún formato.

<video>...</video> Inserta el video que se indica en la etiquita source.
<source src="..." type="..."> Se indica el nombre del archivo o ruta + el formato (video/mp4).
width="..." Ancho en px
height="..." Alto en px
controls añade controles de reproducción.
autoplay Se reproduce al cargar la página.
loop Se reproduce en bucle.
muted Se reproduce sin sonido.

*Se pueden añadir varios <source src="..." type="..." > con diferentes formatos por si el navegador no reconoce algún formato, eligirá siempre el primer formato reproducible. También se puede incluir un mensaje de error antes de cerrar etiqueta </video> para que se muestre si no se puede reproducir ningún formato.

<map>...</map> Convierte imagen en area clicable. Se usa despues de <img>
name="..." Define el nombre el mapa clicable
<area>...</area> Define el area clicable y sus atributos.
alt="..." Titulo del area.
coords="..." Coordenadas del area.
download="..." (Nombre del archivo) Indica que se va a iniciar una descarga.
href="..." URL a redirigir al clicar
target="..." Define el metodo de apertura de la URL.
shape="..." Define tipo de forma.

<canvas>...</canvas> Permite dibujar graficos, requiere JavaScript.
id="..." Define el identificador para utilizarlo en JS.

<svg>...</svg> Contenedor para dibujos SVG definidos en las siguientes etiquetas y atributos.
width="..." Ancho en px del contenedor
height="..." Alto en px del contenedor
<circle cx="..." cy="..." r="..." stroke="..." stroke-width="..." fill="..."/> Dibuja circulo con las caracteristicas de los atributos.
<rect width="400" height="100" style="fill:rgb(0,0,255);stroke-width:10;stroke:rgb(0,0,0)" /> Dibuja rectangulo con las caracteristicas de los atributos.
<polygon points="100,10 40,198 190,78 10,78 160,198" style="fill:lime;stroke:purple;stroke-width:5;fill-rule:evenodd;" /> Dibuja poligono con los vertices marcados en points (X,Y) y el estilo (en este caso Estrella)

<figure>...</figure> Enmarca un documento tipo foto, diagrama, etc. Suele ir seguido del <img>
<figcaption>...</figcaption> Define la descripción de la figura enmarcada.

Links

<a>...</a> Habilita el texto encapsulado para ser clicado y enlazar a otra página. Suele ir acompañado de atributos.
href="..." URL a redirigir. También puede lincar para enviar correo con "mailto:EMAIL_ADDRES" y "subject=ASUNTO". También puede lincar a otra parte de la pagina con el "#ID"
hreflang="..." Idioma de la página redirigida.
download="..." (Nombre archivo) Habilita descarga de un archivo.
rel="..." Relación de la página actual a la redirigida
target="..." Metodo para abrir la página redirigida: _blank (nueva pestaña) _self (misma pestaña [por defecto]) _parent _top

<nav>...</nav> Sirve para encapsular panel de navegación con multiples hipervinculos <a>

Tablas

<table>...</table> Tabla, crea tabla con las siguientes etiquetas
<caption>...</caption> Descriptción, define la descripción de la tabla
<th>...</th> Celda de cabezera, util para titulos de las columnas
<tr>...</tr> Fila, crea una filera nueva.
<td>...</td> Celda, crea una celda dentro de la fila.
colspan="..." (2, 3, 4...) Indica cuantas columnas debe ocupar dicha celda.
rowspan="..." (2, 3, 4...) Indica cuantas filas debe ocupar dicha celda.
<thead>...</thead> Encabezados, define el título de las columnas, útil para agrupar encabezados th.
<tbody>...</tbody> Cuerpo de la tabla, útil para agrupar las celdas del cuerpo de la tabla.
<tfoot>...</tfoot> Pie de la tabla, puede agrupar varia celdas.
<colgroup>...</colgroup> Conjunto de columnas, útil para dar un mismo formato a varias columnas.
<col> Define las propiedades de las columnas de <colgroup>, cada <colgroup> refiere a cada columna en orden.

Formularios

<form>...</form> Formulario, crea formulario con los siguientes atributos y etiquetas.
action="..." (URL, # [misma web]) Acción al finalizar formulario , donde se envia la info escrita en el formulario.
method="..." (GET [envia datos con info de origen], POST [envia datos con orgien oculto])
name="..." Nombre del formulario
autocomplete="..." (on, off) Habilita o deshabilita autocompletado.
target="..." (_blank, _parent...) Donde mostrar siguiente ventana.

<label>...</label> Título del siguiente </input>
name="..." Nombre del campo a rellenar, útil para relacionarlo con el input.

<input> Campo a rellenar por el usuario, permite varios atributos
type="..." (text, button, checkbox, radio, number, mail, password, submit...) Tipo de dato a esperar por el usuario.
id="..." Identificador para este campo, útil para relacionar la info que recibimos cuando enviamos formulario.
name="..." Nombre del campo a rellenar, útil para relacionarlo con el label.
autocomplete="..." (on, off) Habilita o deshabilita autocompletado.
autofocus El navegador te dirige al este campo al cargar la página.
size="..." Número de caracteres aceptados en el campo a rellenar.
minlenght="..." mínimo de carácteres permitidos.
maxlength="..." máximo de carácteres permitidos.
min="..." valor mínimo permitido para numero o fechas.
max="..." valor máximo permitido para numero o fechas.
disabeld="..." Dehsabilita este campo al cargar la página.
placeholder="..." Descripción del input esperado, visible antes de escribir.
enctype="..." Cuando tenemos <input type="data"> define que archivos son aceptados para poder enviar form.
required Obligatorio, campo obligado para continuar.

*Si tenemos varios inputs type="radio" o "checkbox" con el mismo name="...", solo deja seleccionar uno de ellos.

<textarea>...</textarea> Campo para textos extensos, permite redimensionar caja.
cols="..." Define numero de columnas de textarea, donde 1 columna = 1 caracter
rows="..." Define numero de filas de textarea.

<button>...</button> crea boton, útil para hacer boton de "Enviar" formulario
type="..." (submit) define boton para finalizar formulario.
type="..." (reset) Resteablece todas las cajas del formulario.
onclick="..." Define acción a realizar al pulsar el boton. Permite código JS.
<select>...</select> Desplegable, define un seleccionar de varias opciones.
<option>...</option> Opción del desplegable.
selected="..." selecciona dicha opción por defecto.
value="..." valor de dicha opción al enviar formulario.
<datalist>...</datalist> Sugerencias a mostrar en una caja de texto, util para buscador con <input list="NOMBRE">
id="..." Identificador que debe tener mismo NOMBRE que <input list="NOMBRE">
<option>...</option> Sugerencia a mostrar cuando se escribe.

<fieldset>...</fieldset> Agrupa varios inputs relacionados entre ellos.
<legend>...</legend> Descripción de las cajas agrupadas.

<output>...</output> Muestra resultado de operación definida con los valores recibidos en los inputs. Requiere definir atributo oninput para asignar operación a calcular.

Otros

<script>...</script> Código Javascript, permite añadir código JS. Dentro de esta etiqueta no se puede escribir HTML (excepto en document.write(...))
<embed>...</embed> Permite añadir contenido externo a la página.
<var>...</var> Define una variable.
<object>...</object> Define container para recurso externo, útil para incrustar plug-ins.
data="..." (URL o nombre del archivo) archivo o enlace a incrustar.
<param> Define parametros del objeto.

<iframe>...</iframe> Sirve para incrustar otros elementos fuera de la página, otras webs, gmaps, youtube.
src="..." (URL o nombre del archivo) archivo o enlace a incrustar.
width="..." Ancho en px
height="..." Alto en px
