Todo en una misma página

En esta página encontrarás todo el material necesario para consultar mientras te formas como programador Frontend o incluso si eres ya uno experimentado.

No te dejes llevar por la emoción del título de la web, esto es una recopilación de apuntes y documentación de varias tecnologias que se ha ido creando a medida que se aprendian.

Lenguajes disponibles

HTML

CSS

Cascade Style Sheet

Establece la presentación del contenido proporcionando estilos y animaciones.

SASS

Dota a CSS más herramientas y funcionalidades.

TS

Variación de Javascript que aporta tipado para mejorar su escritura y programación.
